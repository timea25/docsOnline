Class VariableStore

Namespace
BOOSEapp
Assembly
BOOSEapp.dll

This class stores and manages all variables used in the BOOSE program. It supports integer variables, real decimal variables and arrays. The class uses a Singleton pattern, meaning only one instance exists and is shared across the application. It allows variables to be created, updated, retrieved and cleared and also provides support for array operations such as poke and peek. This class acts as the central storage for all program data.

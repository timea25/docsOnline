Class AppArray

Namespace
BOOSEapp
Assembly
BOOSEapp.dll

This class represents a custom array used in the BOOSE interpreter. It can store either integer or real decimal values depending on how it is created. The class allows values to be added using the Poke method and returned using the Peek method. It also checks that the index is valid to prevent errors.

AppArray(int, bool)

Creates a new array with a fixed size, the size decides how many values the array can store. The isReal value decides if the array stores integers or decimal numbers. All elements are given a default value: 0 for integers and 0.0 for decimals. This makes sure the array is ready to use.

Peek(int)

Gets a value from the array at a given index. The value is returned as a double, even if it is stored as an integer. The method checks if the index is valid. If not, it throws an error to prevent invalid access.

Poke(int, double)

Stores a value in the array at a given index. If the array stores integers, the value is converted to an integer. Otherwise, it is stored as a decimal value. The method checks if the index is valid. If not, it throws an error to stop invalid access.

Class AppWhileTests

Namespace
BOOSEapp.Tests

BOOSEappTests.dll

This class contains unit tests for while loop behaviour in the BOOSE interpreter. It checks that while blocks are handled correctly by the Parser especially when the condition is false. The tests focus on ensuring that while blocks are skipped when the condition is false, nested while loops are handled correctly, empty while blocks do not cause errors. These tests help confirm that loop control works correctly in the system.

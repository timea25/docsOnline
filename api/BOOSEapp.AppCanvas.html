Class AppCanvas

Namespace
BOOSEapp
Assembly
BOOSEapp.dll

This class represents the drawing area used in the BOOSE program. It is responsible for drawing shapes, lines, and text on the screen. The class controls the pen position, meaning where drawing starts, and also manages the pen colour and thickness. It provides different methods that allow the program to draw shapes such as circles and rectangles, draw lines between points, move the pen without drawing, and write text on the canvas. This class connects BOOSE commands to the visual output.
